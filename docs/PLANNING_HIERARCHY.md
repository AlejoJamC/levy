1. VISION/STRATEGY
   └─ Product Vision Document (todo el producto)

2. INITIATIVE
   └─ Multi-quarter, high-level business goal
   
3. EPIC ⭐ (AQUÍ ESTAMOS)
   └─ Large body of work (1-3 months)
   └─ Ejemplo: "Component 1: Client/Proxy Layer"
   
4. FEATURE
   └─ Deliverable capability within Epic (2-4 weeks)
   └─ Ejemplo: "REST API Endpoint for Query Submission"
   
5. USER STORY / PBI (Product Backlog Item)
   └─ Single unit of work (1-5 days)
   └─ Ejemplo: "As a developer, I can POST /query with JSON payload"
   
6. TASK / SUB-TASK
   └─ Technical implementation steps (hours to 1 day)
   └─ Ejemplo: "Create Pydantic model for LevyRequest"
   
7. SPIKE (para investigación técnica)
   └─ Time-boxed research (1-3 days)
   └─ Ejemplo: "Research best gRPC framework for Python"
